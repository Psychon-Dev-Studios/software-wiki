Free Chat Clients



Psychon Development Studios has released TWO chat clients available to the public. See below for a list of features, which are different between the two editions of the client.

Client →

Feature ↓		 "Free" "Enhanced"
Basic Chat Yes Yes
Limited Access No - Public Yes - UUIDs
Server Controls No Yes
User Status Display No Yes
Customizable Yes Yes - More!
Automatic Login Yes Yes
System Log No Yes - S+L
User Administration No Yes
Spam Protection Unreliable Yes


As you can see, the Enhanced version of the client has more features. In addition, the Enhanced version is the only version being updated, so the Free version will be rapidly falling behind, and eventually may not be compatible with the required server software.
